== Description ==

** AS OF OCTOBER 2014 THIS PLUGIN IS NO LONGER SUPPORTED OR FUNCTIONAL **

This plugin displays your aggregated and moderated social media (Twitter, Facebook, Google+) stream from Crowd Convergence. http://www.crowdconvergence.com

Crowd Convergence is a real-time aggregation and moderation platform empowering users to gain complete control over social media content and integrate into limitless digital media outlets. With Crowd Convergence anybody can collect relevant real-time content from Twitter, Facebook, Google+ and Instagram, moderate through advanced automatic and manual filters and stream to websites, big screens, projection or graphic playback systems.

Before streaming moderated content to your WordPress website you'll need to sign up for an account at Crowd Convergence. After that you can set up a stream, add search terms and start moderating.
Copy the Crowd Convergence WordPress URL from the “links” section of your account and paste it into the options for the Crowd Convergence WordPress plugin.
Moderated content will start to flow through to your WordPress site.

For questions, comments, or feedback, please contact us via email info@crowdconvergence.com or Twitter at @CrowdConverge.


== Installation ==
It is best to install this plugin directly from your WordPress site. If manual installation is required, please make sure to put all of the plugin files in a folder named “crowdconvergence” in the plugin directory, then activate the plugin through the ‘Plugins’ menu in your WordPress site .

== Frequently Asked Questions ==

= I don't have a Crowd Convergence account, can I still use this plugin? =
No. You'll first have to go to http://www.crowdconvergence.com and signup for an account. After that you can set up a product, add search terms and start moderating.
Copy the Crowd Convergence WordPress URL from the “output” section of your product and paste it into the options for the Crowd Convergence WordPress plugin.
Moderated content will start to flow through to your WordPress site.

= How do I filter and moderate my stream? =
Login to your Crowd Convergence account.
Select your product from the dashboard and click "manage”. 
Click the "Approve" or "Reject" button to modify an item’s status.
Changes may take a few seconds to display in the WordPress widget.

= When does the stream update on my page? =
New content will be displayed each time the page is refreshed providing new content has been approved on your Crowd Convergence account.
